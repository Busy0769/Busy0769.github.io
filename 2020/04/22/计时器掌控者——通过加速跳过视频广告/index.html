计时器掌控者——通过加速跳过视频广告

发表于 2020-04-22 | 分类于 技术教程

不管是腾讯视频、爱奇艺、优酷等视频，播放视频都有前n秒的视频广告，本次推荐的插件是通过控制网页计时器，达到加速广告视频的效果，进而达到跳过广告视频的目的。

安装方法

  1. 安装Tampermonkey（油猴），点击到达油猴官网
  2. 安装油猴后，安装“计时器掌控者”插件，点击下载

插件使用方法

  1. 网页左边有悬浮的能量球，鼠标移动过去有加速选项，你可以选择加速或减速，最上面的悬浮球表明现在的倍速
  2. 加速跳过某广告后，需要点击最后一个悬浮球恢复默认速度
  3. 该插件支持所有的网页限时广告的跳过，如腾讯，爱奇艺，优酷等等。
  4. 快捷键操作：’ctrl‘+‘=’ 或 ’ctrl‘+’>’ : 倍率+2, ’alt‘+‘=’ 或 ’alt‘+‘>’ : 倍率x4, ’ctrl‘+‘-’ 或 ’ctrl‘+‘<’ : 倍率-2, ’alt‘+‘-’ 或 ’ctrl‘+‘<’ : 倍率/4, ’ctrl‘+‘0’ 或 ’alt‘+‘0’ : 倍率恢复为1
  5. 在播放视频时，使用该插件可以对视频快进或慢放（注意必须时HTML5视频播放器，Flash播放器是无效的哦）
